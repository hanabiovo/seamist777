# seamist777
# 这是什么

灵感之海是一个帮你管理灵感和待办事项的小工具。

它的核心理念很简单：你脑子里冒出的任何念头，都可以先扔进来——不用想它有没有用，不用想现在能不能做。让它在海里漂着，等你准备好了再捞上来。

# 四个区域

整个 app 由四个区域组成，用底部导航栏切换。

 **海** | 灵感之海 — 所有念头的起点，随手记录，自由漂流
 
 **船** | 我的小船 — 当下专注的事，从海里捞上来放在这里
 
 **记** | 航海日记 — 完成的事情，按时间轴记录你走过的航程
 
 **礁** | 沉默礁石 — 暂时放弃的念头，潮水也许会把它们带回来
 

# 基本用法

## 把念头扔进海里

打开 app，在顶部输入框输入你的想法，按回车或点「入海」。

_例：「学一下 Figma」「买菜别忘了香菜」「写那篇一直拖着的文章」_

念头会先漂在水面上。过了你设定的时间之后，它会自动沉入海底——这是一种温和的提醒：这个念头还没被你关注到。

## 捞起来，开始做

在灵感之海里，点某条念头右边的「上船」，它就会移动到我的小船。

建议船上不要放太多东西——专注一两件，做完再说。

## 完成，记入日记

在我的小船里，点「靠岸」，这条记录就会进入航海日记，附上完成时间。

你也可以直接在航海日记里手动记录刚刚完成的事，不一定要经过「海→船→日记」这条路。

## 随机打捞

如果你很久没看某些念头，它们会沉入海底。点灵感之海右上角的「随机打捞」，会随机把一个沉底的念头重新带到水面。

_这是一个惊喜机制，有时候你会发现一个几周前扔进去、已经忘记的好想法。_

## 搁置

如果一个念头现在明确不想做，点「搁浅」，它会去到沉默礁石。

在沉默礁石里，你可以「放回海里」让它重新漂浮，或者「清除」永久删除。

# 航海日记

航海日记里的记录按时间轴排列，以日期为单位分组。

每一条记录旁边会显示具体的完成时间。

这不只是一个待办清单——它是一份真实的记录，证明你确实做了些什么。

_有时候我们会觉得自己「什么都没做」，翻开日记往往会发现并非如此。_

# 个性化设置

长按底部导航栏任意位置，即可唤出设置面板。

## 主题与颜色

内置五套预设主题：

·  纸本 — 默认米白配色，温暖如纸张

·  夜墨 — 深色模式，适合夜间使用

·  苔绿 — 低饱和绿调，清新

·  暮色 — 深紫调暗色

·  暖沙 — 偏暖的米色调

选完预设主题后，还可以单独调整六个颜色变量：背景、正文、次要文字、淡化文字、极淡文字、分割线。点右侧的色点即可唤出系统拾色器。

## 沉没时间

念头在水面漂浮多久后沉入海底，可以自己设定：

·  30 分钟

·  1 小时（默认）

·  24 小时

·  自定义 — 输入任意分钟数

沉没时间只影响「水面可见性」，念头本身不会消失，随时可以打捞。

# 数据导出

设置面板底部有「导出数据」功能，支持选择导出范围：全部数据、或单独某个区域。

## JSON 备份

导出完整的原始数据，包含所有时间戳和状态信息。

用途：数据备份、迁移、或以后的导入恢复功能。

## Markdown 日记

导出为可读的纯文本格式，航海日记保留时间轴结构。

用途：粘贴到 Notion、Bear、Obsidian 等笔记工具，或直接分享给朋友。

# 安装到手机桌面

灵感之海支持安装为独立 app，安装后全屏打开，没有浏览器地址栏，体验和原生 app 一样。

**Android / Chrome**

·  用 Chrome 打开网址

·  点右上角三个点菜单

·  选「添加到主屏幕」或「安装应用」

**iPhone / Safari**

·  用 Safari 打开网址

·  点底部分享按钮（方块加箭头图标）

·  选「添加到主屏幕」

安装后，数据仍然存储在本地，不需要账号，不联网。

# 关于数据

所有数据都保存在你的浏览器本地（localStorage），不上传到任何服务器。

清除浏览器数据会导致数据丢失，建议定期用「导出 JSON」做备份。
